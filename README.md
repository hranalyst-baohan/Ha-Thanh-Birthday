# Happy Birthday Hà Thanh - Pixel Web

Trang web sinh nhật phong cách pixel art, viết bằng HTML/CSS/JavaScript.

## Cấu trúc

- `birthday_ha_thanh_pixel.html`: file web chính

## Cách chạy trên máy

1. Mở trực tiếp file `birthday_ha_thanh_pixel.html` bằng trình duyệt.
2. Hoặc dùng VS Code + Live Server để xem mượt hơn.

## Up lên GitHub và bật web (GitHub Pages)

1. Tạo repo mới trên GitHub.
2. Đổi tên file chính thành `index.html` (bắt buộc để Pages nhận trang chủ).
3. Upload `index.html` và `README.md` lên nhánh `main`.
4. Vào `Settings` -> `Pages`.
5. Ở mục **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
6. Chờ 1-3 phút, web sẽ có link dạng:
   - `https://<username>.github.io/<repo-name>/`

## Gợi ý

- Nếu muốn giữ tên file hiện tại, bạn có thể tạo thêm `index.html` và copy toàn bộ nội dung từ `birthday_ha_thanh_pixel.html` sang.
- Nên để repo ở chế độ Public để chia sẻ link dễ hơn.
